Selenium Login Test Automation 

This is my beginner Selenium automation project created using Java, Maven, and TestNG.
It automates login testing with valid and invalid test cases.

Project Description
I created this project to practice Selenium WebDriver basics and understand automation testing workflow.
The test covers different login scenarios using mobile/email and password combinations.

Technologies Used
Java
Selenium WebDriver
TestNG
Maven
Eclipse IDE

Test Scenarios Covered
Valid username and valid password
Invalid username and invalid password
Valid username and invalid password
Invalid username and valid password

How to Run the Project
1. Open it in Eclipse as a Maven project
2. Update Maven dependencies
3. Run the TestNG test file
